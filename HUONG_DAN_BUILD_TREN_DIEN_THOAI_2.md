# BUILD TRÊN ĐIỆN THOẠI - 2 CÁCH

## CÁCH 1: Dùng AndroidIDE (build offline trên điện thoại, nhanh nhất)
1. Tải AndroidIDE APK: https://github.com/AndroidIDEOfficial/androidide/releases (file AndroidIDE-v2.8.1.apk)
2. Cài đặt, mở app
3. Bấm Open Project -> chọn folder Vietlott535Sniper đã giải nén
4. Đợi sync, bấm nút Build (hình búa) -> Build APK
5. APK sẽ nằm ở app/build/outputs/apk/debug/

## CÁCH 2: Build online bằng GitHub (không cần cài gì, chỉ cần trình duyệt điện thoại)
1. Tạo tài khoản github.com trên điện thoại
2. Tạo repo mới tên Vietlott535Sniper
3. Upload toàn bộ file trong zip này lên repo (bấm Add file > Upload files)
4. Vào tab Actions -> sẽ thấy workflow "Build Vietlott 535 Sniper APK" đang chạy
5. Đợi 3-5 phút chạy xong -> vào phần Artifacts -> tải file Vietlott535Sniper-APK về
6. Cài APK vào máy

CÁCH 2 là dễ nhất nếu bạn không muốn cài Android Studio trên điện thoại.
