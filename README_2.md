# Vietlott 535 Sniper APK

1. Mở folder này bằng Android Studio Hedgehog trở lên
2. Đợi Gradle sync
3. Cắm điện thoại Android, bật USB Debugging
4. Bấm Run -> sẽ cài app Vietlott 535 Sniper
5. Cấp quyền SMS khi hỏi
6. Build APK: Build -> Build APK(s) -> file ở app/build/outputs/apk/debug/app-debug.apk

Tính năng:
- Cào lotto-8.com/Vietnam/listltoVM35.asp
- Dự đoán 7 thuật toán
- Sniper tự bắn khi jackpot >12 tỷ (kỳ chia 21h hôm sau)
- SMS cú pháp ZLP 535 K1 S 09 14 20 31 35-12 tới 9969

Lưu ý: Xổ số là ngẫu nhiên, app chỉ hỗ trợ thống kê.
